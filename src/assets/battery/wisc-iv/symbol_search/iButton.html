Symbol Search Scoring Key

Administration

Start
Ages 6-7: Symbol Search A Sample Items, Practice Items, then Test Items
Ages 8-16: Symbol Search B Sample Items, Practice Items, then Test Items
Children suspected of intellectual deficiency should be given the item corresponding to their chronological age.

Discontinue
After 120 seconds

Timing

Accurate timing is essential. Begin timing after saying the last word of the instructions. Stop timing when the child completes all the items or 120 seconds have elapsed.

Recording

Prompt Administered (P): Used to record when any subtest-specific prompt is provided  

Scoring

Use the scoring key to score the child's responses.

Subtract Number Incorrect from Number Correct. Total raw scores less than 0 are scored as 0.

Refer to the WISC-IV Administration and Scoring Manual for additional information.
