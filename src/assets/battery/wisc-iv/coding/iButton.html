Administration

Start
Ages 6-7: Coding A Sample Items, then Test Items
Ages 8-16: Coding B Sample Items, then Test Items
Children suspected of intellectual deficiency should be given the item corresponding to their chronological age.

Discontinue
After 120 seconds

Timing

Accurate timing is essential. Begin timing after saying the last word of the instructions. Stop timing when the child completes all the rows or 120 seconds have elapsed.

Recording

Prompt Administered (P): Used to record when any subtest-specific prompt is provided

Scoring

Use the scoring key to score the child's responses.

Score 1 point for each correct response. 

Refer to the WISC-IV Administration and Scoring Manual for additional information.
