Administration

Read each item verbatim to the child.

Start
Ages 6-8: Item 5
Ages 9-11: Item 10
Ages 12-16: Item 12
Children suspected of intellectual deficiency should start on Item 5.

Reverse
Ages 6-16: Score of 0 on either of the first two items given, administer preceding items in reverse order until two consecutive perfect scores are obtained. 

Discontinue
After 5 consecutive scores of 0

Recording

Don't Know (DK): Used to record when the child provides an answer of "I don't know" as his or her only answer

No Response (NR): Used to record when the child does not respond

Prompt Administered (P): Used to record when any subtest-specific prompt is provided 

Repeated Item (R): Used to record when the child asks for an item to be repeated

Query Administered (Q): Used to record when any query is provided

†Teaching: Used to access corrective feedback for 0 point responses on Items 1 and 2

*Query, *Hears Wrong Word, ‡Query, and *Population: Used to access queries for specific sample responses

Scoring

Score 0 or 1 point.

Refer to the WISC-IV Administration and Scoring Manual for additional information.
