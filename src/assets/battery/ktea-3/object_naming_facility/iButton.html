Materials

Client Device (Raised)

Administration

Start
Grades PK–12+: Practice Items, then Trial 1.

Discontinue
Do not discontinue.

Timing

For each trial, begin timing after the last word of instructions, and stop timing after the examinee names the last picture on the screen.

Recording

Don’t Know (DK): Used to record when the examinee provides an answer of “I don’t know” as his or her only answer on the practice items.

No Response (NR): Used to record when the examinee does not respond on the practice items.

Scoring

Each object is worth 1 point. Any object not marked as an Uncorrected Error will be counted as correct. The raw score is based on the total time and total errors for both trials. If there are 20 or more errors, no score can be computed.

When counting the number of errors, accept any reasonable synonyms (e.g., kitten for cat is not an error).

Do not count self-corrections or repetitions as errors.

Uncorrected Error: Occurs when the examinee provides an incorrect response or skips an object, and does not correct the incorrect or skipped response. 

Self-correction: Occurs when the examinee provides an incorrect response or skips an object, but corrects the incorrect or skipped response.

Refer to the KTEA–3 Administration Manual for additional information.
