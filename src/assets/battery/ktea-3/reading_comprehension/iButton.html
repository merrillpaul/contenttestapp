Materials

Client Device (Raised)
Pencil (for Items 102 and 104)
Scratch paper (for Items 102 and 104)

Administration

Start
Grades PK–K: Set A, Items 1–20
Grade 1: Set B, Items 8–35
Grade 2: Set C, Items 21–45
Grade 3: Set D, Items 29–57
Grades 4–5: Set E, Items 39–68
Grades 6–8: Set F, Items 58–86
Grades 9–10: Set G, Items 65–94
Grades 11–12+: Set H, Items 82–105

Reverse
Grades 1–12+: Must pass the first 2 items administered, or drop back one start point. Reverse again as needed. For Sets D–H, administer all items related to the passage before reversing.

Discontinue
After 5 consecutive scores of 0 within a set, or at the end of a set if there are 4 or more scores of 0 within the set.

Decision Point
If there are 3 or fewer scores of 0 in a set, continue to the next set.

Recording

Don’t Know (DK): Used to record when the examinee provides an answer of “I don’t know” as his or her only answer.

No Response (NR): Used to record when the examinee does not respond.

Query (Q): Used to record when any query is provided. 

Repeated Item (R): Used to record when the examinee asks for an item to be repeated or when it appears that he or she was not attending.

Scoring

Score 0 or 1 point.

Refer to the KTEA–3 Administration and Scoring Manuals for additional information.
