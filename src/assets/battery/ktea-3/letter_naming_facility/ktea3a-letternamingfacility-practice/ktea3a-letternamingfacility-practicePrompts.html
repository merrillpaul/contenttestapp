Now I’ll ask you to say the names of some letters. As you point to each letter in the top row, say, What letter is this?

Select each letter the examinee does not name or names incorrectly. Discontinue testing if one or more letters are missed.

If the examinee says the letter sound rather than the letter name, do not count as an error but say, Remember to say the name of the letter. The name of this letter is (insert letter name).

Point to the bottom row and say, Now say the names of these letters. You can use your finger to point if you want. Teach further as needed.
