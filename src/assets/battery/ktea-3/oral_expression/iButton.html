Materials

Client Device (Raised)

Administration

Start
Grades PK–1: Set A, Sample Item A, then Items 1–23
Grades 2–8: Set B, Items 7–28
Grades 9–12+: Set C, Items 13–28

Reverse
Grades 2–12+: Must receive a Task score of 1 on at least one of the first 2 items administered or drop back one start point. Reverse again as needed.

Discontinue
At the item set stop point, or after 5 consecutive items within a set are scored 0 for both Task and Grammar.

Recording

Don’t Know (DK): Used to record when the examinee provides an answer of “I don’t know” as his or her only answer.

No Response (NR): Used to record when the examinee does not respond.

Repeated Item (R): Used to record when the examinee asks for an item to be repeated or when it appears that he or she was not attending.

Self-Correct (SC): Used to record when the examinee self-corrects.

Scoring

Score 0 or 1 point for both Task and Grammar.

The Scoring Rules button provides general guidelines as well as specific examples for each item.

Refer to the KTEA–3 Administration and Scoring Manuals for additional information.
