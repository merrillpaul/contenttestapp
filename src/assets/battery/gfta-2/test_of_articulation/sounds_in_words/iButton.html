Administration

Start
All Items: Item 1

Discontinue
Do not discontinue. Administer all items.

Recording

Substitutions
Select misarticulated phoneme from target word. Select substituted phoneme from IPA keypad. Select the delta symbol if the examinee produces a substituted phoneme other than what is represented on the keypad.

Omissions
Select omitted phoneme from target word. Select the dash symbol below IPA keypad.

Distortions
Phoneme distortions requiring diacritical marks can be recorded in the verbatim handwriting capture area located next to each target word.

Additions
Phoneme additions to target words can be recorded in the verbatim handwriting capture area located next to each target word. 

Scoring

GFTA-2 assesses twenty-three Standard English consonants in all appropriate word positions and sixteen consonant clusters in initial position. Only a single target of each consonant and consonant cluster contribute to the raw score.

Refer to the GFTA-2 Manual for additional information.
